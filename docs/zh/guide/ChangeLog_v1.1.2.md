+ 社区：官网上线 https://mogan.app ，参加由中科院软件所举办的开源之夏2023
+ 界面：图标由GNU TeXmacs开发者之一Massimiliano Gubinelli提供
+ 发布：提供macOS M1/M2的安装包
+ 发布：提供Ubuntu 20.04, Ubuntu 22.04安装包
+ 发布：官网提供实验性的WASM版供体验
