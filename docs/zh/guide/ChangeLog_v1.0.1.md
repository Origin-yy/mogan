# 墨干编辑器V1.0.1
## 什么是GNU TeXmacs
[【科技】GNU TeXmacs：一个科技编辑平台](https://www.bilibili.com/video/BV19741167ik)

## 主要变更
+ 支持GNU/Linux（移植了Philippe Joyez修复的严重bug）
+ Debian平台安装包制作脚本 (`cmake/build_deb.sh`)
+ 输入①②③...⑩的快捷键 ，例如输入`1@`就可以得到①（仅在中文文档中有效）
+ 新增大量示例文档：2021年的十张高考数学试卷和三首诗
